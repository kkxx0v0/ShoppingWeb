# ShoppingWeb

《网络应用开发》课程实验
设计并实现一个电子商务网站的开发和在线部署
application.yaml中包含密码信息暂未开源
已部署：http://8.138.108.248:8642/

【基本功能要求】
顾客：
用户的注册、登录、注销
购买流程（浏览/查询->添加至购物车->付款->发送电子邮件确认收货）
销售：
商品目录的管理（包括最基本的增删改等操作）
后台销售统计报表，销售状态
客户的浏览/购买日志记录
